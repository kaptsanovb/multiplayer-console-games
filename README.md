# Multiplayer console games! (Linux)

## !! WARNING: This is a small hobby project to learn POSIX sockets, the code is bad and proper security considerations have not been made yet !!

Some small multiplayer games made in C with POSIX sockets.  
Currently only Rock-Paper-Scissors has been made!  
(RPS Demo (file))[github.com/kaptsanovb/multiplayer-console-games/blob/main/rpsdemo.mp4]

# Running a game
Clone the repository and compile the relevant game file, e.g.: `gcc rps.c -o rps`.
The game can then be run from the command line.
